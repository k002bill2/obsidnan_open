# 새 노트 생성 워크플로우

## 단계

1. **노트 정보 수집**
   - 제목 확인
   - 저장 디렉토리 결정
   - 태그 선정

2. **표준 템플릿 적용**
   - frontmatter 생성 (created, tags)
   - 기본 섹션 구조 적용

3. **파일 생성**
   - 적절한 디렉토리에 파일 생성
   - 파일명: 제목 그대로 (한국어 가능)

4. **관련 노트 링크**
   - 관련 노트 검색
   - wikilink로 연결

## 명령어

```bash
# Custom Command 사용
/new-note
```
